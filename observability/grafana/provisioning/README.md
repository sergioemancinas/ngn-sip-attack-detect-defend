# Grafana Provisioning

Current scope:

- `datasources/clickhouse.yml`
- `datasources/prometheus.yml`
- `dashboards/dashboards.yml`
- `dashboards/D1_sip_overview.json`
- `dashboards/D2_attack_timeline.json`
- `dashboards/D3_suricata_detection.json`
